jQuery smart-calendar plugin

A jQuery date and time plugin which support mouse-wheel and keyboard control

Demo Code :

From To 

$('#date').smartCalendar({
    format : 'yyyy/MM/dd HH:mm:ss',
    max : {
        caculate : function(){
            return $('#date2').smartCalendar('getTime');
        }
    }
});

Set start to now

Only time :

$('#time').smartCalendar({
    format : 'HH:mm:ss',
	time : new Date(2011, 2, 20, 14, 0, 0),
	step : {
		second : 5
	}
});
